# Bonterra OAuth2 Take-Home

OAuth2 and API security take-home assessment implementation.

## Development

Install dependencies:

```bash
npm install
```

Run the test suite:

```bash
npm test
```

Run TypeScript checks:

```bash
npm run typecheck
```

## Scope

Part 1 will implement a self-contained JWT validation module with typed validation errors, JWKS caching, RS256 signature verification, and claim validation.
